# Customer_shopping_behavior_Analysis
This project analyzes customer shopping behavior using transactional data from approximately 3,900 purchases across different product categories. The goal is to understand customer spending patterns, product preferences, and subscription behavior. The analysis helps identify insights that can support better business and marketing decisions.

Dataset

The dataset contains customer transaction information such as:
Customer ID
Age and Gender
Product category and item purchased
Purchase amount (USD)
Shipping type
Discount applied
Subscription status
Previous purchases
Payment method
Review rating
This data helps analyze purchasing trends and customer segments.

Tools

Python (Pandas, NumPy, Matplotlib/Seaborn) – Data loading, cleaning, and EDA
SQL (PostgreSQL / MySQL / SQL Server) – Data querying and analysis
Power BI – Interactive dashboard creation
Excel – Initial data exploration
Gamma – Presentation creation

Steps

Loaded the dataset using Python.
Performed data cleaning and preprocessing.
Conducted Exploratory Data Analysis (EDA) to understand trends.
Ran SQL queries to analyze customer behavior and revenue patterns.
Built an interactive Power BI dashboard to visualize insights.
Created a project report and presentation summarizing key findings.

Dashboard

The Power BI dashboard provides visual insights such as:
Revenue by product category
Customer demographics
Purchase patterns and trends
Shipping preferences
Impact of discounts on purchases

Results

Key insights from the analysis include:
Certain product categories generate higher purchase frequency.
Discounts influence customer purchasing behavior.
Repeat customers show different buying patterns compared to new customers.
Shipping preferences vary across customer groups.

How to Run

Download or clone the repository.
Open the Python notebook (.ipynb) to run data analysis and EDA.
Use the SQL file to execute queries in your SQL environment.
